HTML

HTML - Hyper Text Markup Language, It`s a markup language with the objective of structuring web pages (it`s the skeleton of the web page).
As a markup language, HTML utilizes tags to identify content.

Elements

A HTML document is made of elements, an element is identified by a start tag, an end tag and the content between them.
Some elements don`t have an end tag (like <br>).
