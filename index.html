HTML

HTML significa Hyper Text Marckup Language (Linguagem de Marcação de Hyper Texto), é uma linguagem de marcação que tem como objetivo estruturar web pages.
Como uma linguagem de marcação, HTML usa tags para identificar conteúdos.

Elementos

Um documento HTML é composto por elementos, um elemento é composto por uma tag de inicio e uma de fim, e o conteúdo entre elas. Alguns elementos não possuem tag de fim.

<html>... </html>

A tag html abre e fecha o código HTML.

<head>... </head>

A tag head vem logo após a tag html, e indica o cabeçalho da página, onde se encontram todos os elementos não visuais da página que à ajudam a funcionar.

<body>... </body>

Dentro da tag body ficam todos os elementos visuais e estruturais da página, títulos, paragráfos, listas e imagens são alguns dos elementos que ficam dentro da tag body.

<title>... </title>

Dentro da tag title (que deve estar localizada dentro da tag head), vai a descrição da página que ficará localizada na aba do navegador. O título também é usado em mecânismos de pesquisa.

<p>... </p>

A tag p representa paragráfo, os navegadores automaticamente adicionam um linha vazia antes e uma depois de um paragráfo.

</br>

Single Line Break, pula para a linha de baixo, não tem tag de fim.

<hr />

Horizontal Line, cria uma linha horizontal, não tem tag de fim.

<!--... -->

Comment, comenta o texto dentro da tag.

Elementos de Formatação

<b>... </b>

Formata o texto dentro da tag para negrito.

<big>... </big>

Aumenta a fonte do texto dentro da tag.

<i>... </i>

Formata o texto dentro da tag para itálico.

<small>... </small>

Diminui a fonte do texto dentro da tag.

<strong>...</strong>

Formata o texto dentro da tag para negrito e marca o mesmo como importante.

<sub>... </sub>

Formata o texto dentro da tag para subscrito.

<sup>... </sup>

Formata o texto dentro da tag para sobrescrito.

<ins>... </ins>

Formata o texto dentro da tag para sublinhado.

<del>...</del>

Formata o texto dentro da tag para riscado.

Heading (<hn>... </hn>)

Transforma o texto em um título, o título não muda somente a formatação, um título é usado por mecanismos de pesquisa para estruturar páginas. Existem seis níveis de título:

<h1>... <h1>

<h2>... <h2>

<h3>... <h3>

<h4>... <h4>

<h5>... <h5>
<h6>... <h6>
